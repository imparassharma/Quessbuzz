How do you find the missing number in a given integer array of 1 to 100?

Suppose I am taking 153 for an example
First calculate the cube of its each digits
1^3 = (1*1*1) = 1
5^3 = (5*5*5) = 125
3^3= (3*3*3) = 27
Now add the cube
1+125+27 = 153
It means 153 is an Armstrong number.

How to check if two strings are Anagram?

To check given strings are anagram or not for that purpose we have to match each character of the string. And also we have to count the occurrence of each characters from both the strings. To implement it first we will arrange both strings in the same order. And then compare each character of both string position wise. If all positions have the same character in both strings then it means strings are anagram otherwise it is not an anagram.
